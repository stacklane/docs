Private Files

Any file prefixed with an underscore _fileName.html will never be publicly accessible, and is only used for server side includes or similar logic (for example SASS import or Mustache partial).

Similarly any files whose direct parent folder has an underscore such as /_files/something.html, will be considered private even if the files themselves are not prefixed with an underscore.
