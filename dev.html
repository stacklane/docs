Development

Stacklane doesn't require any special UI setup to start coding -- everything needed to run a Stacklane site is in standard file formats, hosted in a GitHub source repository. Branch and tag to your heart's delight. We recommend using WebStorm and its Stacklane plugin.

IDE Plugins

WebStorm

+ Stacklane Plugin

WebStorm natively supports all Stacklane file formats, including GIT integration. Grab the Stacklane plugin for WebStorm for easy testing.

Download
